# NestJS

## Installing nestJS

``` cmd
# using npm 
npm install -g @nestjs/cli

# Using Yarn
yarn global add @nestjs/cli
```

## Generating Components using CLI

``` cmd
# Creating a new application 
nest new training-tracker-api
```

## Running the project
``` terminal
# run with watch enabled
yarn start:dev
npm start:dev

# run without watch
yarn start
npm start
```
