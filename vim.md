## Add line number to vim 
``` cmd
:set number
```

## Add color syntax to vim
``` cmd
: syntax on
```

## Change color scheme for syntax
``` cmd
:colorscheme slate
```

## cmd to list all installed color schemes in vim 
``` cmd
 ls -l /usr/share/vim/vim*/colors/
```

## Save Contents in vim profile
open the profile file
``` cmd
vim ~/.vimrc
```

``` cmd                                                                                                                 
   :set number
   :syntax on
   :colorscheme slate
```
